## Auto-Save Feature Description:
- After enabling this feature, the switch status will be saved locally.

## Auto-Load Feature Description:
- The auto-load feature is only activated when "<font color=#FF0000>*</font>" is marked after the switch.

## Position Detector Feature Description:
- This feature detects the player's position and resets the timer if movement is detected.

## Re-Enter Auto-Load Script Feature Description:
- Once enabled, the script will auto-load upon re-entering the game.
